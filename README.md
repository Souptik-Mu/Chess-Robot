# Chess-Robot

after running the program detect.py set the camera.

After cam setup press 'x' to select the corners. (a new window will pop up for this)
in the new window, select 4 points clockwise manner
the frame will be cropped and transformed to the shape.

Press 'c' to get current bord position reading in console.

Press 'q' to exit.

[ to re-select the square feild, tap 'x' and close the new popped window.     Now repeat the step to select corners. ]
